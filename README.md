# ✈️ Manifest TXT Converter App

Aplikasi untuk mengubah file manifest penerbangan (.txt) menjadi:
- Tabel penumpang
- Summary otomatis
- Export Excel

## 🚀 Fitur
- Upload file .txt
- Parsing otomatis
- Summary (Male, Female, Infant, Bagasi, Berat)
- Download Excel

## ▶️ Cara Menjalankan
```bash
streamlit run app.py
